# 🎮 CodeQuest — Learn to Code by Playing

A coding **game**, not a lecture. You get bite-sized, interactive challenges with
instant feedback, a score, and a streak that ramps up the difficulty as you go.
Way less reading — way more doing.

Built with **Next.js (App Router)**, **TypeScript**, and the **Claude API**
(via the official Anthropic SDK). Every challenge is freshly generated by Claude,
so you never run out.

## The four challenge types

| Type                | What you do                                             |
| ------------------- | ------------------------------------------------------- |
| 🔢 Multiple choice  | Pick the right answer from a few options.               |
| 🖨️ Predict output   | Read a tiny program and guess what it prints.           |
| ✍️ Fill in the blank | Type the missing token to complete the code.            |
| 🐞 Find the error    | Click the one buggy line in a short program.            |

Pick a track (Python, JavaScript, HTML & CSS, or "Surprise me"), then play. Get it
right and your streak — and your points — grow. Miss one and you see the answer
plus a one-line explanation before the next challenge.

## Getting started

### 1. Install dependencies

```bash
npm install
```

### 2. Add your Anthropic API key

```bash
cp .env.example .env.local
```

Edit `.env.local` and set your key (get one at
[console.anthropic.com](https://console.anthropic.com/)):

```
ANTHROPIC_API_KEY=sk-ant-...
```

### 3. Run it

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and start playing.

## Configuration

| Variable            | Default           | Description                          |
| ------------------- | ----------------- | ------------------------------------ |
| `ANTHROPIC_API_KEY` | _(required)_      | Your Anthropic API key.              |
| `ANTHROPIC_MODEL`   | `claude-opus-4-8` | Override the Claude model if needed. |

## How it works

```
Browser (app/page.tsx)
  │  POST /api/challenge  { track, level, recent }
  ▼
Next.js route (app/api/challenge/route.ts)
  │  client.messages.create(...)  ← Anthropic SDK
  ▼
Claude (claude-opus-4-8)  ──►  one challenge as JSON  ──►  rendered as a playable card
```

- The game tracks your **level** (rises as you solve more) and the **concepts**
  you've already seen, and sends both to Claude so challenges get harder and don't
  repeat.
- The challenge "level designer" instructions live in
  [`lib/challengePrompt.ts`](lib/challengePrompt.ts) — tweak them to change the
  style, difficulty curve, or topics.

## Project structure

```
app/
  api/challenge/route.ts   # Generates one challenge (JSON) from Claude
  layout.tsx               # Root layout + metadata
  page.tsx                 # Game loop: track picker, score/streak, next
  globals.css              # Game styling
components/
  ChallengeCard.tsx        # Renders + grades all four challenge types
lib/
  challengePrompt.ts       # The "level designer" instructions for Claude
  types.ts                 # Challenge shape + validation
```

## Deployment

Deploys cleanly to any Node host that supports Next.js (e.g. Vercel). Set
`ANTHROPIC_API_KEY` in the host's environment variables, then:

```bash
npm run build && npm start
```
